## ✅ Successes
These are the things I’ve been able to accomplish so far:

- Built and configured an Ubuntu VM in VirtualBox  
- Installed tools like Wazuh and learned how they work  
- Captured and analyzed local network traffic using Wireshark  
- Learned how to dissect packets layer by layer  
- Understood TCP handshakes, flags, and application‑layer behavior

## ❌ Failures (and What They Taught Me)
I include my failures because they’re part of the process:

- Misconfigured my VM networking more than once  
- Broke my Ubuntu install and had to reinstall  
- Misread packet flags until I learned how TCP actually works  
- Tried capturing traffic on the wrong interface  
- Confused encrypted vs. unencrypted traffic in Wireshark  
- Spent hours troubleshooting simple mistakes  
- Overlooked basic settings that caused bigger problems  

Each failure pushed me to understand the tools better instead of giving up.
